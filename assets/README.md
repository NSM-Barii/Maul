# Assets Folder

This folder contains visual assets for the MAUL project.

## Files

### banner.svg
Animated SVG banner with:
- Pulsing red glow effects (lightsaber aesthetic)
- Animated scan line
- Infrastructure mapping feature badges
- Cyberpunk/Sith theme

### darth_maul.png (Required)
Download a high-quality transparent PNG of Darth Maul and save it here as `darth_maul.png`

**Recommended sources:**
- https://www.pngmart.com/image/tag/darth-maul
- https://www.kindpng.com/free/darth-maul/
- https://www.pngall.com/darth-maul-png/

**Specs:**
- Format: PNG with transparent background
- Recommended width: 600-1000px
- Keep file size under 2MB for fast loading

Once you add `darth_maul.png`, it will display under the banner in the README.
